# LOOMICON
> 专为设计师与开发者打造的**免费商用服装图标库**

Loom Icon 是一个专注于服装的矢量图标资源站，提供高质量、像素完美的图标素材，所有图标均可免费用于个人或商业项目。

## 🚀 快速使用
### 方式1：直接下载
1. 访问 [LOOMICON 官网](https://aglt2w.github.io/loomicon/)
2. 搜索或浏览分类找到目标图标
3. 点击图标下方 **下载按钮**，获取 SVG 文件
4. 将 SVG 文件直接嵌入项目，或转换为所需格式

### 方式2：复制 SVG 代码
1. 找到目标图标，点击 **复制按钮**
2. 直接粘贴 SVG 代码到 HTML 中使用
3. 可自定义图标颜色、大小等样式

## 🛠️ 部署指南
本项目基于 HTML/CSS/JavaScript 开发，使用 GitHub 管理代码，Vercel 一键部署，也可部署到其他平台。

### 部署到 Vercel
1. Fork 本仓库到你的 GitHub 账号
2. 登录 [Vercel](https://vercel.com)，导入 Fork 后的仓库
3. 无需额外配置，直接点击 **Deploy** 完成部署
4. （可选）绑定自定义域名：在 Vercel 项目设置中添加阿里云购买的域名

### 本地开发
1. 克隆仓库
   ```bash
   git clone https://github.com/你的用户名/loom-icon.git
   ```
2. 进入项目目录，直接打开 `index.html` 即可运行
3. 修改图标资源：将新的服装 SVG 图标放入指定目录，更新图标数据配置

目前已覆盖以下分类（持续更新中）：
- 服装品类：上衣、裤子、裙装、外套、内衣等
- 制衣工具：剪刀、针线、缝纫机、尺子等
- 面料纹理：棉、麻、丝、毛、牛仔等
- 配饰部件：纽扣、拉链、口袋、腰带、帽子等
- 穿搭场景：休闲、正式、运动、礼服等
